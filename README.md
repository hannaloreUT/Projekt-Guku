# Projekt-Guku

Meie projektiks on mäng “Kivi-paber-käärid”, kus inimene saab Pythoni programmiga mängida. Mängule lisame ka graafilise poole (milline see välja hakkab nägema ei ole veel kindel). Otsustasime teha just mängu, mitte näiteks kooli asjadega abistavat programmi, sest mäng on lõbusam ja nii saame olla natuke loovamad ja teha projekti enda jaoks põnevamaks.
